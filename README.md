## Features

- **Real-time Conversion** — Instantly converts pixels to inches, cm, and mm.
- **Smart Presets** — One-click presets for Mugs, A4, Instagram, YouTube, T-Shirts, etc.
- **Quality Health Check** — Automatic warnings for low DPI or insufficient resolution.
- **Light and Dark Mode** — Professional theme system with system preference detection.
- **Fully Responsive** — Works seamlessly on desktop, tablet, and mobile.
- **SEO Optimized** — Schema markup, semantic HTML, and featured snippet ready.
- **Zero Dependencies** — No jQuery, no frameworks. Just pure HTML, CSS, JS.
- **Privacy First** — All calculations happen client-side. No data collection.
