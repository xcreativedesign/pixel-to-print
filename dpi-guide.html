Quick Answer

DPI (Dots Per Inch) is a measure of print resolution indicating how many ink dots a printer places in one linear inch. 300 DPI is the standard for high-quality prints like photos and brochures, while 72 DPI is only for screens.

What is DPI? (Detailed Explanation)

Think of it this way: if you have a 1-inch square printed at 300 DPI, your printer places 300 dots horizontally and 300 dots vertically, creating 90,000 tiny dots in that square inch. More dots mean smoother gradients and sharper detail.

DPI vs PPI: What's the Difference?

While often used interchangeably, DPI and PPI refer to different things. Understanding the distinction helps you communicate accurately with print shops and designers.

DPI - Dots Per Inch

  • Refers to printed ink dots
  • Used by printers and print shops
  • Physical output measurement
  • Affects print quality directly

PPI - Pixels Per Inch

  • Refers to screen pixels
  • Used in digital design software
  • Digital resolution measurement
  • Affects screen display sharpness

Practical Note: In most design workflows, when you set an image to "300 DPI" in Photoshop, you're technically setting 300 PPI. However, the print industry commonly uses "DPI" to describe both concepts, so don't worry too much about the terminology in day-to-day work.

DPI Standards by Print Type

Different print applications require different DPI levels. Here's a comprehensive guide to what DPI you should use for various print projects.

Print Type Recommended DPI Notes
Photos & Art Prints 300+ Maximum detail for close viewing
Magazines & Books 300 Industry standard for publications
Brochures & Flyers 300 Sharp text and images
Business Cards 300-350 Small text needs high resolution
Sublimation (Mugs, Shirts) 300 Essential for vibrant transfers
Newspapers 150-200 Lower quality paper absorbs ink
Posters (Small) 200-300 Viewed from moderate distance
Banners & Signs 100-150 Large format, viewed from far
Billboards 15-50 Very large, extreme viewing distance

Calculating Print Size from Pixels

The formula for converting pixels to inches is straightforward but essential for every designer to know.

The Formula

Inches = Pixels ÷ DPI

Example 1: High Quality Print

3000 px ÷ 300 DPI = 10 inches

Example 2: Large Banner

3000 px ÷ 100 DPI = 30 inches

Common DPI Mistakes to Avoid

Mistake: Resampling to "Increase" DPI

Simply changing the DPI number in Photoshop without resampling doesn't add detail—it just changes the print size. To truly increase resolution, you need more pixels, not just a higher DPI number.

Mistake: Using Screen Images for Print

Images saved at 72 DPI for web use will appear pixelated when printed. Always use the original high-resolution file, not the web-optimized version.

Mistake: Over-Up scaling Small Images

Upscaling a 500px image to 3000px won't create detail that wasn't there. The result will be blurry, not sharp. Start with high-resolution source images.

Mistake: Ignoring Viewing Distance

Using 300 DPI for a billboard is unnecessary and creates massive file sizes. Match your DPI to the viewing distance of the final product.

Pro Tips for Best Results

Start High, Scale Down

Always design at the highest resolution you might need. You can always reduce resolution for smaller prints, but you can't add detail later.

Use Vector When Possible

Logos, text, and illustrations should be vectors (SVG, AI, EPS). They scale infinitely without quality loss at any DPI.
